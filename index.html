Easy Twitter Boostrap Modal

Hello

This is a class to help you take Twitter Bootstrap Modal to its full power, and have a easy way to do anything with it

This class is exceptional if you're planning to make a full AJAX Application

Main Popup Trigger

LayoutHelperPopup.Modal(title, content, footer, after_load, large_popup)

Available Functions

LayoutHelperPopup.ShowErrors(errors);
LayoutHelperPopup.Modalfunction(title, content, footer, after_load, large_popup);
LayoutHelperPopup.GetModalObject();
LayoutHelperPopup.ModalClose();
Params :
  • Title
    • This is the title of the popup
  • Content
    • This is the content of the popup
  • Footer
    • This is the footer of the popup
    • Available params : false, true, string.
      • If this is set to true, the footer is included.
      • If this is set to false, the footer will be ignored, not added
      • If this is a html string, the html will be placed in the footer
  • After_load
    • This is the action triggered after the popup has been rendered
    • Get the popup jquery object using : LayoutHelperPopup.GetModalObject() and use find to navigate
  • Large Popup
    • This has 3 options, true, false, string
      • If this is set to true, the popup will be 90% of the screen.
      • If this is set to false, popup will be normal
      • If this is a html string, the popup will receive the width from this variable

Examples

Simple Usage
Code Behind
LayoutHelperPopup.Modal('Title', 'Simple Modal');
Simple Modal Large
Code Behind
LayoutHelperPopup.Modal('Title', 'Simple Modal Large', true, false, true);
Simple Usage With Event Fired
Code Behind
        LayoutHelperPopup.Modal('Title',
                                'Simple Modal With fired event after render <a class="click_me">Click Me</a>',
                                true, 'firedEvent', false);
    function firedEvent() {
            LayoutHelperPopup.GetModalObject().find('a.click_me').bind('click', function(event){
              event.preventDefault();

              $(this).html('This was clicked !');
              $(this).unbind('click');
            });
          }
Simple Usage With Event Fired And Custom Footer
Core Behind
LayoutHelperPopup.Modal('Title', 'Simple Modal With fired event and custom footer',
                                '<button class="btn" data-dismiss="modal" aria-hidden="true">Close</button>'
                                + '<button class="btn btn-primary some_trigger">Click Me</button>', 'firedEventFooter', false);
        
function firedEventFooter() {
          LayoutHelperPopup.GetModalObject().find('.some_trigger').bind('click', function(event){
            event.preventDefault();

            LayoutHelperPopup.Modal('Title', 'Some Event Triggered');

            $(this).unbind('click');
          });
          }
